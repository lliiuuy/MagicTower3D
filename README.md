# **MagicTower3D**
using Opengl to rebuild the Magic Tower in 3D

#### **1. 项目简介** ####
----------
本项目基于opengl库，使用C++语言+GLFW库编写，将一个经典RPG游戏魔塔（日文名：魔法の塔， 英文名：Magic Tower）还原并投影到3D世界中，实现第一人称游玩体验。
