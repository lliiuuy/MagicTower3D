#### 项目简介 ####
------
MagicTower3D是一个基于C++、OpenGL与GLFW库， 使用Visual Studio开发的应用程序，目的在于还原原版2D魔塔（日文名：；英文名：Magic Tower/ Tower of the Sorcerer）的同时，将游戏投影到三维世界中，实现第一人称的游戏体验
